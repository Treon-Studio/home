Keputusan untuk mempelajari Next.js secara mendalam menjadi salah satu titik balik penting dalam perjalanan teknis kami. Sebelumnya, kami menggunakan berbagai framework dan pendekatan yang berbeda untuk setiap proyek web. Next.js menawarkan solusi lengkap dengan server-side rendering, static site generation, API routes, dan ekosistem yang matang â€” semuanya dalam satu framework.

Proses pembelajaran dimulai dari dokumentasi resmi, diikuti dengan membangun beberapa proyek kecil untuk memahami konsep-konsep seperti App Router, Server Components, dan data fetching patterns. Kami juga aktif mengikuti perkembangan komunitas Next.js dan berbagi catatan pembelajaran secara internal. Setiap anggota tim diminta untuk membangun setidaknya satu proyek personal menggunakan Next.js.

Setelah beberapa bulan intensif mempelajari dan menggunakan Next.js, kami memutuskan untuk menjadikannya sebagai framework standar untuk seluruh proyek frontend di Treon Studio. Keputusan ini memberikan konsistensi teknis, mempercepat onboarding developer baru, dan memudahkan sharing knowledge antar tim proyek.
